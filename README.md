# Leaner Software

> Implementation for the [leaner.software](https://leaner.software) web app.

## Development

```shell
npm install
```
Gets dependencies.

```shell
npm start
```
Launches the web server.

```shell
npm run lint
```
Executes eslint checks for the `src` folder.

```shell
npm test
```
Executes all available unit and e2e tests, including eslint checks.
